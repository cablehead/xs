<TabItem label="nushell">
```nushell
cargo install cross-stream --locked
use xs.nu *
xs serve ./store
```

![nushell .cat output](../../assets/01-nu.png)

```nushell
"a quick note" | .append notes
.cat
.cat | last | .cas  # retrieve the last note
```

![nushell output](../../assets/02-nu.png)
</TabItem>

<TabItem label="bash">
```sh
cargo install cross-stream --locked
xs serve ./store
```

![bash output](../../assets/01-bash.png)

```sh
echo "a quick note" | xs append ./store notes
xs cat ./store
xs cat ./store | tail -n 1 | jq -r .hash | xargs xs cas ./store  # retrieve the last note
```
![bash output](../../assets/02-bash.png)
</TabItem>
